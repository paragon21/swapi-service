## Тестовое SPA на API Swapi
Демо: 
### Перед установкой требуется предварительно установить модули
```
git clone https://github.com/paragon21/swapi-service
cd swapi-service  
npm i --save
```
### Далее запускаем dev-сервер
```
npm start
```
### Для сборки проекта используется команда
```
npm run build
```
